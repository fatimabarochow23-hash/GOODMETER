# 论文图例分析功能需求清单
## 为 GOODMETER 升级「插件医生」模式的功能规格

> 本文档由 Claude 整理于 2026-04-29，对应蔡艺扬毕业论文《动画电影中能量碰撞音效的分层构建与效果链应用》所需的图例素材。

---

## 一、设计目标

将 GOODMETER 的响度表扩展为一套**音频分析对比工具**，使其能够像 PluginDoctor 一样可视化频谱/相位/时间结构，且支持**导入用户素材做 A/B 对比**，直接截图作为论文插图。

---

## 二、核心需求按优先级

### 🔴 P0 — 必须要有（缺了论文没法插图）

#### 1. A/B 双轨对比模式

**用于**：所有"处理前 vs 处理后"对比图（饱和、Disperser、共鸣滤波等）

**功能要求**：
- 加载两个音频文件（原素材 + 处理后的素材）
- 或：加载一个音频文件即时通过选定插件处理，实时对比处理前后
- 显示模式：上下分屏 / 左右分屏 / 透明叠加（三种可选）
- 同步播放、同步缩放、同步时间轴
- 截图输出格式：PNG，保留屏幕上所有信息

**论文用途**：

| 图4.2.3/4.2.4 | 瞬态素材→Disperser前后 → 同上 |

#### 2. 频谱分析（FFT）——带频率分区标注

**用于**：展示频率分层、谐波结构、频段分布

**功能要求**：
- FFT 频谱显示（可调 FFT size: 1024/2048/4096/8192）
- 叠加多条频谱迹线（不同颜色）
- **频率分区标注**：在频谱顶部或底部用色带/竖线标注：
  - 低频区 20-200Hz（红色阴影）
  - 中频区 200-2000Hz（绿色阴影）
  - 高频区 2000Hz+（蓝色阴影）
- 可叠加谐波序列标记线（基频+倍频虚线）

**论文用途**：
| 图号 | 用途 |
|---|---|
| 图3.1.1 频率分层图 | 用三段色带标注低频/中频/高频，配上三层素材各自的频谱迹线 |
| 图4.2.1 谐波生成 | 100Hz的基频+2次/3次/4次谐波处自动画虚线标记 |

### 🟡 P1 — 重要（提升论文技术可信度）

#### 3. 相位分析模式

**用于**：Disperser 全通滤波器的核心论证——"改变相位关系而非频率幅度"

**功能要求**：
- 同时显示两条曲线：
  - **幅度响应**（dB vs Frequency）——应为平直，证明"幅度不变"
  - **相位偏移**（Degrees vs Frequency）——应为非线性曲线，证明"相位改变"
- 叠加显示方式：幅度用左Y轴（蓝色），相位用右Y轴（红色）
- 频率轴对数刻度 20Hz-20kHz
- **特别适合截图的角度**：两条迹线画在同一张图上时视觉冲击力最强——读者一眼就能看到"幅度是平的，相位是斜的"

**论文用途**：
| 图号 | 用途 |
|---|---|
| 图4.2.2（重截） | Disperser全通滤波器——幅度平直 + 相位偏移曲线。**比当前linear频谱图更有说服力** |

#### 4. Spectrogram / 时频谱

**用于**：展示时间维度上的频率变化——CST结构、瞬态散布

**功能要求**：
- 频率-时间-幅度三维显示（热力图）
- 可调时间分辨率
- 可选择色板（至少支持：热力/灰度/彩虹）
- 支持在时频谱上标注时间区间（C-S-T区间标记）
- 支持导入素材与处理后素材的**并排时频谱对比**

**论文用途**：
| 图号 | 用途 |
|---|---|
| 图3.2 CST时间结构图 | 选一个典型的Charge-Shot-Tail音效，在时频谱上标出三个时间区段 |
| 图4.2.3/4.2.4（重截） | 瞬态素材过Disperser前后的时频谱对比——看到"能量沿时间轴拉散" |

### 🟢 P2 — 锦上添花（有更好，没有也不影响核心论证）

#### 5. 波形显示 + 时间标注

**用于**：展示瞬态结构、CST 时间分段、Attack/Release

**功能要求**：
- 高精度波形显示（可缩放至样本级）
- 时间轴标注/标尺
- 可添加垂直标记线（标记瞬态峰值点、衰减转折点）
- 可导出带时间标注的波形截图

**论文用途**：
图3.2 CST时间结构图的辅助视图

#### 6. 谐波/频谱峰值标注

**用于**：自动识别并标注频谱中的峰值频率

**功能要求**：
- 自动检测频谱峰值，显示峰值的频率和dB值
- 对基频自动标记倍频位置（2f, 3f, 4f...）
- 支持手动添加标注

**论文用途**：
图4.2.1 谐波生成——自动标注 100Hz, 200Hz, 300Hz, 400Hz

#### 7. 三维瀑布图（Waterfall）

**用于**：展示共鸣滤波器的衰减特性

**功能要求**：
- 频率-时间-幅度三维频谱
- 可旋转视角
- 可调整时间范围

**论文用途**：
图4.2.3 共鸣滤波器族——展示滤波器共鸣峰在时间上的衰减形态

## 三、截图层面的规范要求

为了论文插图统一，**所有截图应当**：

1. **背景色**：白色（论文印刷要求）
2. **线条和文字颜色**：黑色 + 区分度高的辅助色（红/蓝/绿仅用于标注）
3. **字体**：无衬线字体，字号足够在论文中可读（约10-12pt）
4. **坐标轴标注**：包含单位（Hz, dB, ms, degrees）
5. **图例**：多迹线叠加时必须有图例
6. **导出格式**：PNG，72dpi，宽度统一 13.5cm（论文版心宽度）
7. **文件名命名规范**：`{章节}_{功能}_{描述}.png`，如 `4.2.1_saturation_100Hz_comparison.png`

## 五、技术实现建议（面向 GOODMETER 开发）

### 架构
- 直接在 `Source/` 下新增一个 `Analyser/` 子模块
- 核心分析引擎与 GUI 显示分离
- 使用 GOODMETER 已有的 `dsp/` 库中的 FFT 和滤波器模块

### 最小可行功能（MVP）
1. **单轨频谱分析**（已有类似功能，扩展 FFTSize 和显示选项）
2. **A/B 双轨加载**（新增：处理两个 AudioBuffer 的并行显示）
3. **相位曲线绘制**（新增：FFT 结果中提取相位数据，画第二条曲线）
4. **截图导出**（已有：GOODMETER 的 GUI 截图功能可以复用）

### 建议的开发顺序
1. 单轨频谱 + 分区标注 → 图3.1.1 频率分层图（1天）
2. A/B 对比模式 → 图4.2.1/4.2.5 优化（2天）
3. 相位分析模式 → 图4.2.2 重截（1天）
4. Spectrogram 模式 → 图3.2 CST、图4.2.3/4.2.4 优化（2天）
5. 共鸣滤波器对比 → 图4.2.6（1天）
