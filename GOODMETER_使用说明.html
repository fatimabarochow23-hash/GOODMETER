1. 概览

GOODMETER 是一款专业音频计量插件，提供九块独立的测量面板，覆盖响度合规、频谱分析、 立体声场监测、瞬态动态等核心维度。所有计量面板均支持折叠/展开、拖拽排序与紧凑模式， 适配从母带缩混到直播推流的全场景工作流。

插件提供三种格式：

  • VST3 — 适用于 REAPER、Cubase、Studio One、Ableton Live 等
  • Audio Unit (AU) — 适用于 Logic Pro、GarageBand、Final Cut Pro 等
  • Standalone App — 独立运行，无需 DAW

2. 安装与卸载

安装

双击 GOODMETER_Installer_v1.0.0.pkg 安装包，按照引导完成安装。安装包已通过 Apple 公证 (Notarization)，可直接打开，无需绕过 Gatekeeper。

安装路径：

  • /Library/Audio/Plug-Ins/VST3/GOODMETER.vst3
  • /Library/Audio/Plug-Ins/Components/GOODMETER.component
  • /Applications/GOODMETER.app

安装完成后，重启 DAW，在插件列表中即可找到 GOODMETER。

卸载

将以上三个路径中的对应文件移至废纸篓即可。

3. 界面总览与模式切换

GOODMETER 采用可折叠卡片布局，九块测量面板以多列瀑布流排列，自适应窗口宽度。

两种显示模式

模式 特点 列数断点
Mini 模式（默认） 紧凑标题栏 (24px)、更小间距，适合嵌入混音工作流 ≥600px → 3列 / ≥400px → 2列 / 其他 → 1列
Normal 模式 标准标题栏 (48px)、更大间距，适合全屏专注分析 ≥1000px → 3列 / ≥650px → 2列 / 其他 → 1列
切换方法：在 NONO 机器人身体区域鼠标右键双击即可在 Mini / Normal 模式之间切换。

卡片折叠/展开

点击任意面板的标题栏即可折叠或展开该面板。折叠状态下只显示标题，节省纵向空间。

窗口支持自由缩放，面板会自动适应可用空间。展开的面板平均分配列内剩余高度。

4. LEVELS

响度表

核心响度监测面板，提供峰值柱与 LUFS 数据读数，是混音/母带阶段确保响度合规的主力工具。

峰值柱 (Peak Bars)

  • L / R 双通道，量程 -60 dB ~ 0 dB
  • 渐变色带：绿色（安全）→ 黄色（-18 dB 以上）→ 粉红（-6 dB 以上）
  • 峰值保持标记（黑色竖线），保持 1 秒后缓慢衰减
  • 目标响度参考线（荧光橙色虚线），逼近过载时脉冲闪烁

响度标准选择

点击 LEVELS 卡片标题栏右侧的下拉菜单，选择目标响度标准：

Streaming-14.0 LUFS（通用流媒体）
EBU R128-23.0 LUFS（欧洲广播标准）
ATSC A/85-24.0 LUFS（美国广播标准）
Netflix-27.0 LUFS
YouTube-14.0 LUFS
抖音 (Douyin)-14.0 LUFS
哔哩哔哩 (Bilibili)-16.0 LUFS

LUFS 数据区

指标含义
Momentary瞬时响度 (400ms 窗口)
Short-Term短期响度 (3s 窗口)
Integrated积分响度（全曲/全段平均，含 EBU R128 双门限）
LU Range响度范围 (LRA)，反映动态变化幅度
True Peak L / R真峰值（dBTP），超过 -1.0 dBTP 时高亮警告
超过目标响度的指标会以粉红色高亮显示，方便快速定位超标项。

5. VU METER

VU 表

经典模拟 VU 表，以弧形表盘 + 指针的方式反映音频的 RMS 电平，具有类似硬件的惯性阻尼特性。

表盘说明

  • 量程：-30 dB ~ +3 dB
  • 绿色区域：-30 dB ~ 0 VU（安全范围）
  • 红色区域：0 VU ~ +3 VU（过载警告）
  • 刻度层级：主刻度（-30, -20, -10, 0）、中刻度（-5, -3, -1, +1, +2, +3）、细分刻度（每 1 dB）

指针运动模拟真实 VU 表的惯性响应（约 300ms 积分时间），适合观察音乐的整体动态走势而非瞬态峰值。

6. 3-BAND

三频段

三频段能量分布可视化，以三种不同形状的"炼金术玻璃容器"直观显示低/中/高频的实时电平。

频段划分与容器形态

频段频率范围容器形状颜色
LOW20 Hz ~ 250 Hz球形烧瓶 (Beaker)粉红
MID250 Hz ~ 2 kHz量筒 (Graduated Cylinder)黄色
HIGH2 kHz ~ 20 kHz锥形瓶 (Erlenmeyer Flask)绿色

液面高度与对应频段能量成正比。液面表面有弯月面高光，容器外围有淡淡的霓虹光晕。

7. SPECTRUM

频谱

FFT 频谱分析仪，实时显示 20 Hz ~ 20 kHz 的频率分布，适合定位频率冲突与谐波问题。

技术参数

  • FFT 大小：4096 点（75% 重叠，~43 Hz 帧率）
  • 频率轴：对数刻度，20 Hz ~ 20 kHz
  • 幅度轴：-100 dB ~ +6 dB
  • 渐变填充：顶部粉红，底部渐隐
  • 频率网格标注：20, 50, 100, 200, 500, 1k, 2k, 5k, 10k, 20k Hz

8. PHASE

相位

相位相关表，以"冷凝器"科学仪器造型展示 L/R 通道的相位相关性 (-1.0 ~ +1.0)。

读数解释

数值区间含义颜色
+1.0完全单声道（L = R）青色
0.0完全不相关中性
-1.0完全反相（L = -R），可能导致单声道回放时信号消失粉红

液滴在玻璃螺旋管中的位置直观反映当前相位状态。相关性为正时液滴显示为青色，为负时变为粉红色。

注意：如果相位值长期偏向 -1.0（粉红区域），请检查信号链中是否存在相位反转问题， 尤其在多麦克风录音或并联处理的场景中。

9. STEREO

立体声场

立体声场分析面板，由左侧 LRMS 试管与右侧 Goniometer（李萨如图）组成。

LRMS 试管（左侧）

试管含义颜色
L左声道电平粉红
R右声道电平粉红
M中声道 (Mid = L + R)黄色
S侧声道 (Side = R - L)绿色

左侧有 dB 刻度尺（-6 ~ -60 dB），三条刻度线标注在 25%、50%、75% 液位处。

Goniometer / 李萨如图（右侧）

菱形背景中显示实时李萨如轨迹，用于直观观察立体声宽度与声像分布。

  • M 轴（纵向）：能量集中在中间说明单声道成分多
  • S 轴（横向）：能量偏向两侧说明立体声宽度大
  • 轨迹偏向左上 = 声像偏左，偏向右上 = 声像偏右
  • 内外两个菱形参考框帮助判断能量范围

10. SPECTROGRAM

瀑布图

实时瀑布图（Waterfall），以颜色深浅显示频率-时间-能量的三维分布，适合追踪随时间变化的频率事件。

参数

  • 频率范围：30 Hz ~ 20 kHz（对数刻度）
  • 动态范围：-80 dB ~ -10 dB
  • 颜色映射：白色（安静）→ 粉红色（中等）→ 深红色（响亮）
  • 滚动方向：从左向右，实时推进
  • 频率标注（右侧）：50, 100, 200, 500, 1k, 2k, 5k, 10k, 20k Hz

瀑布图使用独立的后台渲染线程与环形缓冲区（2048 帧），确保大窗口下也不会卡顿。

11. PSR

瞬态比

Peak-to-Short-Term Ratio（峰值瞬态比），反映音频的动态保留程度，是评估压缩处理是否过度的关键指标。

计算公式

PSR = Peak (dBFS) - Short-Term LUFS

读数解释

PSR 值含义
> 12 dB动态充裕，瞬态保留优秀
8 ~ 12 dB常规混音范围
< 8 dB过度压缩警告（红色虚线标注）

面板中央显示镜像波形图，实时绘制约 6.7 秒的 PSR 历史。右上角数字读数：正常为青色，低于 8 dB 时变为粉红色。

12. NONO

智能伴侣

NONO 是 GOODMETER 的"赛博炼金术"机器人伴侣，集离线音频分析与交互动画于一身。 它能独立分析音频文件的响度指标，无需加载到 DAW 中播放。

外观组成

  • 球形主体：金属质感壳体，带镜面高光与边缘反射
  • 面罩（正面）：深色屏幕，蓝色荧光边框
  • 眼睛：电蓝色光条，跟随音频电平张合
  • 全息耳：固态底座 + 青色能量刃片
  • 漂浮试管：粉红色液体，跟随音频节奏晃动
  • 反重力辉光：底部青色光晕

鼠标操作一览

操作位置效果
左键双击 NONO 身体（正面） 翻转到背面（显示"+"拖放区）
左键双击 NONO 身体（背面） 翻转回正面
左键单击 NONO 身体（背面） 打开文件选择器，选择音频文件进行分析
左键双击 NONO 身体（结果显示状态） 清除分析结果，试管倾倒倒液动画，回到正面
左键双击 漂浮试管 进入搬家模式（面板拖拽排序）
左键双击 NONO 身体（搬家模式中） 退出搬家模式
右键双击 NONO 身体（任意状态） 切换 Mini / Normal 显示模式
拖放音频文件 NONO 身体（任意状态） 自动翻转到背面并开始分析

音频文件分析

NONO 可以分析离线音频文件，独立计算完整的 EBU R128 响度指标。有两种触发方式：

  1. 拖放：将音频文件直接拖到 NONO 身上
  2. 文件选择：双击翻到背面，单击"+"区域打开文件选择器

支持的音频格式：.wav .mp3 .aiff .aif .flac .ogg

分析结果

分析完成后 NONO 自动翻回正面，下方气泡显示结果：

指标含义
峰值文件真峰值 (dBFS)
瞬时最大最大瞬时响度 (400ms 窗口)
短期最大最大短期响度 (3s 窗口)
平均响度积分响度 (EBU R128 双门限)
中声道中声道 (Center) 积分响度 — 仅多声道 (≥6ch) 文件显示
多声道支持：NONO 完整支持多声道音频文件（最多 64 通道），遵循 ITU-R BS.1770-4 标准的声道权重： LFE 通道自动排除，环绕/顶部通道 +1.5 dB 加权。多声道文件分析结果右上角会显示通道数标签（如 "58ch"）。
中声道 (Center) 响度：当分析 5.1 或更多声道的音频文件时，NONO 会额外计算第 3 通道（Center）的独立积分响度， 以粉红色显示在结果气泡第三行。这对检查对白/人声的响度非常实用。

特殊动画

  • 分析中：蓝色涟漪扩散动画
  • 过载爆炸（峰值 ≥ 0 dBFS）：试管粉碎 → 玻璃碎片飞溅 → 液体溅射 → 3 秒后自动修复
  • 倒液清除：试管倾斜 120° 倒出液体，液面下降，然后恢复直立
  • 晕眩：快速拖拽/缩放窗口时 NONO 的眼睛会变成旋转蚊香图案
  • 眨眼：搬家模式下悬停即将交换时 NONO 会眨眼确认
  • 编辑模式变色：试管液体从粉红变为霓虹绿，液面冒泡

13. 搬家模式 — 面板排列自定义

搬家模式允许您通过拖拽来自由调整九块面板的显示顺序，打造个人专属布局。

操作流程

步骤操作
1. 进入搬家模式 左键双击 NONO 的漂浮试管
2. 观察状态变化 所有面板开始轻微抖动，试管液体变为霓虹绿色并冒泡
3. 拖拽面板 按住任意面板拖动，面板会变为半透明（60% 不透明度）
4. 悬停交换 将面板拖到目标位置上方悬停 1 秒，NONO 眨眼确认可以交换
5. 松开完成交换 松开鼠标，两块面板以平滑动画完成位置交换
6. 退出搬家模式 左键双击 NONO 的身体，或左键双击任意空白区域
搬家模式下面板的折叠/展开功能暂时禁用，避免操作冲突。退出搬家模式后恢复正常。
